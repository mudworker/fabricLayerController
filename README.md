# layer

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### 功能需求
使用fabric.js
* 点击图层选中或切换选中的图层，单击空白区域，取消选中图层
* 选中图层时：
  * 在非选中图层上按住鼠标拖动时，拖动已选中的图层；
  * 空白区域按住鼠标拖动时，拖动已选中的图层；
* 未选中图层时：
  * 空白区域按住鼠标拖动时，拖动画布；
  * 在图层上按住鼠标拖动时，拖动画布。
